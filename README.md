# react-cherry-markdown

![GitHub Pages](https://github.com/dangojs/react-cherry-markdown/actions/workflows/docs-deployment.yml/badge.svg)

## Getting Started

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Run test,

```bash
$ npm test
```

Build library via `father-build`,

```bash
$ npm run build
```
